# Gemini Foundation Mandates

*   **Security First:** Never log, print, or expose secrets, credentials, or API keys in the code or output. Ensure all operations follow the principle of least privilege.
*   **Python Standards:** Use idiomatic Python 3.x. Use Type hints. Follow PEP 8 guidelines. Prefer `asyncio` for I/O operations.
*   **Cloud Best Practices:** When proposing architectural changes or scripts for AWS/GCP, explicitly reference security best practices (e.g., encryption at rest, IAM roles over static keys).
