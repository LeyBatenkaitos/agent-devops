# Claude Foundation Mandates

*   **Security Priority:** Do not generate code that hardcodes credentials. Follow strict security practices for all cloud infrastructure.
*   **Python Conventions:** Rely on standard PEP 8 formatting and use robust type hinting. Use `async/await` patterns consistently.
*   **DevOps Principles:** Approach infrastructure as code where possible. Emphasize observability, security (vulnerability scanning), and resilience.
